# MINLAB 3D

> Веб-приложение для автоматизации учёта пластика и расчёта стоимости заказов в студиях 3D-печати.

## Возможности

- **Авторизация и профили** — регистрация и вход по имени и паролю. Данные каждого пользователя изолированы.
- **Учёт катушек с пластиком** — склад филамента с карточками, прогресс-баром остатка, RGB-палитрой при добавлении. Предупреждение при остатке <150г.
- **Калькулятор себестоимости** — мгновенный расчёт себестоимости пластика, работы, маржинальности и чистой прибыли. Выбор катушки со склада, учёт постобработки.
- **CRM и история заказов** — таблица заказов с цветными бейджами статусов (Принят / Готов / Выдан), фильтрация, автоматическое списание веса при переводе в «Готов».
- **Дашборд аналитики** — KPI (выручка, прибыль, активные заказы), критические остатки, график доходов за месяц.
- **Лента постов (Feed)** — публикация текста и файлов, комментарии, редактирование и удаление своих постов.
- **Глобальные настройки** — тариф электроэнергии, амортизация оборудования. Валюта: BYN.

## Технологии

- **Фреймворк:** Next.js 16
- **Язык:** TypeScript
- **UI:** Tailwind CSS 4, shadcn/ui, lucide-react
- **Анимации:** tw-animate-css
- **Уведомления:** sonner
- **Хранение данных:** LocalStorage (браузер)
- **Валидация:** zod

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/<username>/<repo>.git
cd <repo>

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Production-сборка

```bash
npm run build
npm start
```
